# SportConnect Pro - Guia de Projeto

## 1. Paleta de Cores (Brand Identity)
As cores são focadas em um visual moderno, esportivo e de alta performance.

- **Primária (Verde):** #00C853 (Destaque principal, sucesso)
- **Primária Escura:** #00952A (Botões pressionados e gradientes)
- **Primária Suave:** #E8F5E9 (Fundo de emblemas/badges)
- **Acento (Laranja):** #FF6D00 (Alertas e atenção)
- **Perigo (Vermelho):** #D32F2F (Erros e cancelar)
- **Background:** #F9FAFB (Fundo das páginas)
- **Surface:** #FFFFFF (Cards e containers)
- **Texto Principal:** #1A1A1A (Títulos e conteúdo)
- **Texto Secundário:** #616161 (Descrições)
- **Texto Mudo:** #9E9E9E (Placeholders)

---

## 2. Tipografia e Estilos
Baseada em fontes modernas (Inter/Roboto) com foco em peso visual.

- **Títulos Hero:** Font-Black (Extra Bold), tamanho 32-40px.
- **Títulos de Seção:** Font-Black, tamanho 20-24px.
- **Corpo do Texto:** Font-Medium/Bold, tamanho 14-16px.
- **Badges/Tags:** Font-Black, Uppercase (Todo em maiúsculo), Espaçamento de letra (Tracking) aumentado.

---

## 3. Modelo de Dados (Appwrite Schema)

### Collection: usuarios
- **nome_completo:** string (255)
- **email:** string (email)
- **foto_perfil:** string (url/string)
- **tipo_perfil:** enum (JOGADOR, ORGANIZADOR)
- **nivel_habilidade:** string (INICIANTE, INTERMEDIARIO, AVANCADO, PROFISSIONAL)
- **interesses:** string[] (array de strings)

### Collection: quadras
- **nome_local:** string
- **id_organizador:** string (relacionamento)
- **endereco_completo:** string
- **latitude / longitude:** float
- **status_aprovacao:** enum (PENDENTE, APROVADO, REJEITADO)

### Collection: eventos
- **titulo:** string
- **esporte:** string
- **data_evento:** string
- **preco_por_vaga:** float
- **nivel_requerido:** enum (INICIANTE, INTERMEDIARIO, AVANCADO)
- **status:** enum (ABERTO, LOTADO, CONCLUIDO)

---

## 5. Elementos de UI Premium
- **Canto Arredondado (Rounded):** 24px a 40px (estilo bem arredondado/moderno).
- **Sombras (Shadows):** Suaves (shadow-black/5), simulando profundidade realista.
- **Efeito Visual:** Uso de Backdrop Blur (vidro fosco) em menus flutuantes.
